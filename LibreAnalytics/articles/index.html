Fonctionnalités clés de LibreAnalytics

1. Tableau de bord intuitif et complet

Visualisez en temps réel les performances de votre site avec des graphiques interactifs, des cartes géographiques et des tableaux détaillés :

  • Visites et visiteurs uniques : Suivi précis du trafic.
  • Sources de trafic : Google, réseaux sociaux, liens directs, etc.
  • Géolocalisation : Pays et villes de vos visiteurs.
  • Appareils et navigateurs : Compatibilité et expérience utilisateur.
  • Comportement : Pages vues, temps passé, taux de rebond.

Exemple d’insight automatisé :
"Votre taux de rebond est de 65% sur la page d’accueil. Pensez à optimiser votre contenu ou votre call-to-action pour améliorer l’engagement."

2. Tracking avancé et paramétrable

LibreAnalytics va au-delà du simple comptage de visites :

  • Paramètres UTM : Tracez l’efficacité de vos campagnes marketing.
  • Événements personnalisés : Suivez les clics, les téléchargements, ou toute interaction clé.
  • Sessions uniques : Identification précise des visiteurs (sans cookies).

3. API et intégrations

Accédez à vos données depuis n’importe quel outil grâce à mon API RESTful. Exemples d’utilisation :

  • Intégration avec Google Data Studio, Excel, ou des dashboards custom.
  • Automatisation de rapports via des scripts Python ou JavaScript.
  • Alertes personnalisées (ex: "Si trafic > 1000 visites/jour, envoyer une notification").
Exemple de requête API (JavaScript) : 
fetch(`https://gael-berru.com/LibreAnalytics/smart_pixel_v2/public/api.php?

4. Souveraineté et sécurité

Avec LibreAnalytics, vos données restent sous votre contrôle :

  • Hébergement 100% français, sans transfert vers des serveurs étrangers.
  • Pas de revente de données à des tiers.
  • Chiffrement des données et authentification robuste.
  • Conformité native avec le RGPD et les réglementations européennes.

Comment installer LibreAnalytics ?

En seulement 3 étapes, commencez à tracker votre trafic :

  1. Créez votre compte gratuit : Rendez-vous sur LibreAnalytics et inscrivez-vous avec votre email et le mot de passe de votre choix.
  2. Récupérez votre code de tracking : Après connexion, copiez le script généré automatiquement pour votre site.
    Exemple de code : 
    <script data-sp-id="SP_79747769"
        src="https://gael-berru.com/LibreAnalytics/smart_pixel_v2/public/tracker.js"
        async>
</script>
  3. Installez le script : Collez ce code juste avant la balise `` de votre site. Le tracking commence immédiatement !

Astuce : Pour tester l’installation, visitez votre site et vérifiez que la visite apparaît dans votre tableau de bord en temps réel.

FAQ : Questions fréquentes

Mes données sont-elles vraiment privées ?

Oui. LibreAnalytics est hébergé en France et ne vend aucune donnée à des tiers. Le code est open source et peut être audité par quiconque.

Est-ce que le pixel ralentit mon site ?

Non. Le script pèse seulement 4KB et se charge de manière asynchrone, sans bloquer l’affichage de votre site.

Combien de temps les données sont-elles conservées ?

Les données sont conservées 365 jours pour les plans Gratuit et Pro, et 2 ans pour le plan Business.

Puis-je exporter mes données ?

Oui (plans payants). Vous pouvez exporter vos données en format CSV ou JSON depuis l’onglet "Export".

Le RGPD est-il géré ?

Absolument. LibreAnalytics inclut une gestion native des cookies et permet l’anonymisation des IP pour une conformité totale.
